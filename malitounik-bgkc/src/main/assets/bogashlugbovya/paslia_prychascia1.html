МАЛІТВА ПАДЗЯКІ

Слава Табе, Божа. Слава Табе, Божа.
Слава Табе, Божа.

Дзякую Табе, Госпадзе Божа мой,* што Ты не адкінуў мяне, грэшнага,* але палічыў дастойным быць удзельнікам Тваіх сьвятасьцяў.* Дзякую Табе,* што Ты ўдастоіў мяне прыняць Твае сьвятыя Дары.

Але, чалавекалюбны Ўладару,* што дзеля нас памёр і ўваскрос* і дараваў нам гэтыя страшныя жыватворчыя Тайны* для дабра і асьвячэньня душ і целаў,* дай, каб яны былі і мне для аздараўленьня душы і цела,* для прагнаньня ўсякага супраціўніка,* для прасьвятленьня вачэй сэрца майго,* для супакою душэўных маіх сілаў,* для веры непахіснай,* для шчырай любові,* для спаўненьня мудрасьці,* для захаваньня Тваіх запаветаў,* для павелічэньня Тваёй Боскай ласкі* і для асягненьня валадарства Твайго.

Каб я,* захаваны імі ў Тваёй сьвятасьці,* быў заўжды памятны Тваёй ласкі* і жыў ужо не для сябе,* але для Цябе,* нашага Ўладара і Дабрачынца;* і, закончыўшы гэтае жыцьцё ў надзеі жыцьця вечнага,* дасягнуў непрамінальнага супакою,* дзе безупынна чутны голас сьвятых* і пануе вечная радасьць тых,* што аглядаюць невыказную красу Твару Твайго.

Бо Ты — сапраўднае жаданьне і невыказная радасьць тых, што любяць Цябе, Хрысьце Божа,* і Табе ўсё стварэньне пяе славу праз усе вякі.* Амін.
